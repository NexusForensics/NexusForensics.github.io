What It Does

Make image evidence inspectable, searchable, and explainable.

Lens Witness focuses on visual evidence that needs careful documentation: screenshots, photos, public images, cropped details, text inside images, and investigator annotations.

Inspect details

Zoom into regions of interest, mark details, preserve context, and keep notes tied to the source image.

Read text

Run OCR against screenshots and images so visual text can become searchable case material.

Compare visuals

Document reference images, visible similarities, uncertain details, and reviewer-facing comparison context.

Export package

Create image reports with annotations, crops, thumbnails, OCR text, source notes, hashes, and local file paths.

Workflow

From image capture to witness-ready explanation.

Lens Witness is designed to help an investigator slow down, document the visible facts, and separate evidence from interpretation.

Step 1Add image

Bring in a screenshot, public image, photo, or saved case media.

Step 2Preserve source

Attach URLs, file paths, capture notes, and hash records.

Step 3Mark details

Annotate visual findings, crops, objects, faces, text, or scene details.

Step 4Run OCR

Extract visible text and connect it to the image evidence.

Step 5Export

Package the annotated image record for case review or disclosure.

Visual Findings

Show what was seen, not just what was concluded.

Lens Witness is meant to make visual findings reviewable: the marked area, the original image, the OCR text, the note, the source context, and the evidence package should all point back to each other.
