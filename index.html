What It Does

Turn social platforms into organized case evidence.

Nexus Social Recon follows the investigator's path: preserve the source, capture local evidence, enrich the evidence with OCR and indexing, then build a report that keeps every finding tied to source context.

Capture the source

Save public profiles, posts, comments, replies, images, videos, screenshots, source URLs, and local evidence files while reviewing the page.

Automation

Background work stays visible to the investigator.

Longer tasks can keep running after capture. The report can update as OCR, indexing, media checks, and evidence enrichment complete.

01

OCR and keyword checks

Images, screenshots, comments, bookmarks, and sampled video frames can be processed for searchable text.

Example: a phone number appears inside a screenshot rather than page text.
02

Bookmark enrichment

Saved findings can include thumbnails, local media links, source context, comment liker lists, and report references.

Example: manually fetch a profile avatar only when a comment is bookmarked.
03

Case targets and alerts

Names, handles, keywords, faces, objects, and recurring case signals can be checked against captured evidence.

Example: a known alias appears in a newly captured profile or report.
04

Account match finder

Compare handles and visual clues across platforms to build a documented cross-platform lookup report.

Example: one username family is checked across Snapchat, Instagram, Twitch, and other sources.
05

Processing queues

Background OCR, indexing, and media capture can continue while the investigator reviews the rest of the case.

Example: comment profile photos and media indexing finish after the source capture is saved.
06

Case activity logs

Capture steps, local files, timestamps, source context, reports, and investigator details can be recorded in the case.

Example: show when a source was captured and where local evidence was saved.

Built-In Intelligence

Connect evidence without hiding the evidence trail.

The intelligence layer is built to keep leads explainable. Source links, thumbnails, local paths, comments, and confidence context stay visible.

User index cards

Collect profile details, aliases, account links, notes, extracted emails, media references, and related evidence hits in one reviewable card.

OCR and media search

Extract text from saved images, screenshots, reports, bookmark media, and sampled video frames for searchable case signals.

Face and visual matching

Build a face index for case media and compare selected faces, objects, and details against captured evidence.

Cross-case index

Find repeated identifiers, handles, media hashes, OCR text, and visual targets across a local encrypted evidence index.

Example output: cross-platform lookup with matched profiles, source links, visual scores, and report metadata.
